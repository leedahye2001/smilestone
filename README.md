# 🚀sMarket 기술 공유 플랫폼
- 판매자가 자신의 기술을 공유할 수 있는 플랫폼
- 기술 공유를 위해서 1대1 채팅, 음성 채팅, 화상 채팅 기능을 제공


> ✨각 프로젝트 코드는 각 브랜치에서 볼 수 있습니다.<br>
✨프로젝트 자료는 docs 패키지에서 볼 수 있습니다.

## ✨모듈 별 기능
> 각 모듈 별 기능을 나열하였습니다
* 앱/웹 공통
   - 로그인/가입
   - 홈 (당근마켓과 유사) → 게시글 리스트
   - 검색바
   - 판매화면(글 제목, 가격 입력, 게시글 내용 작성, 사진 첨부)
   - 채팅 (구매자-판매자)
* 앱
    - 1:1 채팅 대화
    - 1:1 전화
* 웹
    - 1:1 채팅 대화
    - 1:N 영상통화
    - 1:N 화면공유(Optional)
